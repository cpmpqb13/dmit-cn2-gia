## FAQ：买之前最常问的问题

### DMIT有退款保证吗？

DMIT提供3天退款政策（部分套餐），购买后如果不满意可以在3天内申请退款。具体条款以购买时官网显示为准，建议下单前确认。

### 一个账号可以开几台VPS？

没有硬性限制，可以在同一账号下购买多个套餐，不同机房、不同线路都可以叠加。

### 客服响应速度怎么样？

工单系统是主要支持渠道，响应时间通常在几小时到一天之内，不是即时在线客服。紧急问题建议工单里标注优先级。

### CN2 GIA和普通CN2有什么区别？

CN2 GIA是电信国际精品网络的最高等级，全程走GIA（Global Internet Access）骨干，拥塞少、延迟低、稳定性高。普通CN2（GT）在高峰期容易出现拥塞，晚高峰延迟可能明显上升。DMIT的Premium系列走的是GIA，这是它定价高于普通VPS的核心原因。

### 香港节点和洛杉矶节点怎么选？

香港节点物理距离近，延迟通常在30-50ms，适合对延迟极度敏感的场景。洛杉矶CN2 GIA延迟在130-170ms，但流量配额更大、价格相对低一档，适合流量需求较大的业务。如果预算允许，香港Premium是延迟最优解；如果要兼顾流量和成本，洛杉矶Pro是更常见的选择。

### 支持Windows系统吗？

DMIT的VPS默认提供Linux系统，部分套餐支持自行安装Windows，但需要自备授权，且内存建议2GB以上。

### 续费价格会变吗？

DMIT的续费价格通常与购买时一致，不会在续费时涨价。但促销期间的特价套餐到期后可能恢复原价，购买前确认续费价格。
